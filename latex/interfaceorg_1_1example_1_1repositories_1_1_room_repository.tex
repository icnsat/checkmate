\doxysubsection{Detailed Description}
Репозиторий для управления сущностями \doxylink{classorg_1_1example_1_1entities_1_1_room}{Room} (Комнаты). 

Содержит методы для работы с комнатами, включая поиск доступных комнат, поиск комнаты по ID и извлечение отеля по ID комнаты.
