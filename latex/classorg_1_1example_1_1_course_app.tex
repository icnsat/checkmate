\doxysubsection{Detailed Description}
Главный класс Spring Boot приложения. 

Этот класс является точкой входа в приложение и содержит\+:
\begin{DoxyItemize}
\item Основную конфигурацию Spring Boot
\item Метод main() для запуска приложения
\item Автоматическую настройку компонентов Spring
\end{DoxyItemize}

\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item @\+Configuration
\item @\+Enable\+Auto\+Configuration
\item @\+Component\+Scan
\end{DoxyItemize}
\end{DoxyNote}
